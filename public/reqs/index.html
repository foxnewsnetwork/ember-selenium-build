Requirements

You'll need to proper install the following

npm install --save chromedriver # "^2.21.2"
npm install --save ember-cli-babel # "^5.1.6"
npm install --save exists-sync # "0.0.3"
npm install --save fs-extra # "^0.26.7"
npm install --save lodash # "^4.10.0"
npm install --save rsvp # "^3.2.1"
npm install --save selenium-webdriver # "^2.53.1"
There is no blueprint to do this because ember-selenium-build's version requirements are loose, whereas yours might be very tight

*note: if you're using this with ember-cli-github-pages or ember-gitlab-pages, be aware you are no longer forced to depend on hash location type!
